# 40_domain 인덱스

이 디렉토리는 도메인 지식(기능 명세, 정책, 용어)을 보관합니다.
AI는 작업 시 이 파일을 먼저 읽고, 관련된 파일만 선택적으로 읽어옵니다.

## 파일 목록

| 파일 | 설명 |
|------|------|
| `policies/common/` | 공통 repo에서 동기화되는 정책 (이 repo에서 직접 수정 금지) |
| `policies/local/` | 이 repo 고유 정책 |
| `specs/` | 기능 명세 문서 모음 |
| `glossary.md` | 프로젝트 도메인 용어 사전 |
